✦ Core Capabilities

Powerful Features

Four interconnected AI modules working together to deliver seamless sign-language-to-speech translation.

🤲

Real-Time Sign Recognition

Hand tracking via MediaPipe combined with a custom CNN trained on ISL A–Z gestures for accurate, real-time letter detection.

😊

Emotion Detection

Facial expression analysis using a FER2013-trained convolutional neural network that classifies 7 distinct emotions.

✨

Intelligent NLP

Auto-correction of signed abbreviations into grammatically perfect sentences using spell-check and language models.

🗣️

Emotion-Aware TTS

Voice output that dynamically adapts pitch, rate and tone based on the detected facial expression for natural-sounding speech.

⚡ Pipeline

How It Works

A streamlined four-step pipeline from gesture to spoken word.

01
✋

Sign a Letter

Hold up an ISL hand sign in front of the webcam. MediaPipe tracks 21 hand landmarks in real time.

02
🧠

CNN Detection

The custom-trained convolutional neural network classifies the detected hand pose into one of 26 ISL letters.

03
💬

NLP Correction

Accumulated letters are processed by the NLP module — spell-checked and formed into coherent sentences.

04
🔊

Speak Output

The emotion-aware TTS engine speaks the corrected sentence with the tone matching your facial expression.
